## 🚀 Featured Projects

Each project is full-stack (Spring Boot + React/TS), tested, CI-verified, Dockerized, and documented.

### 🏦 [NovaBank](https://github.com/naitik0009/novabank) — Digital Banking Platform
A retail banking app built on a **true double-entry ledger** with **ACID-safe, idempotent money transfers**. Financial integrity is enforced by a "conservation of money" test, and a 20-thread concurrency test proves zero double-spend or balance corruption. Stateless JWT auth, role-based access, and an immutable audit log.
<br/>`Java 21` · `Spring Boot 3` · `PostgreSQL` · `JWT` · `React` · `TypeScript` · `Docker`

### 🗂️ [TaskFlow](https://github.com/naitik0009/taskflow) — Real-Time Collaborative Kanban
A Trello-style board with **live multi-user collaboration over WebSocket/STOMP** — collaborators see card moves and edits instantly, no refresh. Features a fractional-position ordering algorithm for drag-and-drop and **role-based access control** (Owner/Member/Viewer) enforced on every endpoint and the WebSocket handshake.
<br/>`Java 21` · `Spring Boot 3` · `WebSocket/STOMP` · `PostgreSQL` · `React` · `TypeScript`

### 🎟️ [TicketHub](https://github.com/naitik0009/tickethub) — Concurrency-Safe Ticketing & Payments
An event ticketing platform engineered to **never oversell under load** — atomic conditional inventory updates + optimistic locking, proven by a 50-thread race test. Adds **idempotent payments**, TTL-based seat holds with a scheduled sweeper, and Redis-backed rate limiting with graceful in-memory fallbacks.
<br/>`Java 21` · `Spring Boot 3` · `PostgreSQL` · `Redis` · `Stripe` · `React` · `TypeScript`
